Instructions

Hello! Welcome to Learnera. Users can create lessons, each lesson is made up of several questions. Anyone can view lessons, however only the owner can edit their own lessons. There are 4 tools available for creating lessons, these include term/definitions, fill in the blanks, multiple choice, and math. Lessons can be combined to make a course, courses can be used to quiz oneself on multiple relating lessons at the same time.

An investment in knowledge pays the best interest.
